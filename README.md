# Digital Banking 2.0

Microservices-based digital banking system built with Java and Spring Boot.

## Architecture

- customer-service
- account-service
- (future) transaction-service
- (future) api-gateway
- (future) auth-service

## Tech Stack

- Java 21
- Spring Boot
- PostgreSQL
- Apache Kafka

## Goals

- Practice microservices architecture
- Event-driven communication
- Clean architecture principles

## Como Rodar
docker compose up -d customer-postgres account-postgres pgadmin
